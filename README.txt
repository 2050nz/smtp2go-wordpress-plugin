== Description ==

The SMTP2GO Wordpress Plugin allows you to use SMTP2GO as the email sender for your Wordpress installation.


== Installation ==

Copy the smtp2go directory into your wp-content/plugins directory. Activate the plugin through the wp-admin plugins page.
